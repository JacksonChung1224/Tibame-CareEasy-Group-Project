## 任務：評估送出前的告知與同意
於 AssessmentQuiz 最後一步（送出／查看結果按鈕之前）插入告知區塊：

**告知文案（一字不差）**：
```
📋 資料使用說明
・蒐集目的：產生您的補助試算結果，並用於改善推估準確度。
・使用資料：您在本問卷填寫的評估選項（不含姓名、身分證等個人識別資料）。
・保存方式：以匿名方式保存於本平台資料庫；您可隨時來信要求停止使用。
・提供第三方：不會提供任何第三方。
・本試算為示範性參考，不是政府正式核定，也不是醫療診斷；
  實際資格與額度須由各縣市長期照顧管理中心評估核定。
```

**同意機制**：
1. Checkbox：`我已閱讀並同意上述資料使用說明`——**預設未勾選**
   （設計文件 5.4 明文要求，不可預勾）。
2. 未勾選時「查看試算結果」按鈕 disabled（同時具視覺 disabled 態與
   HTML `disabled` 屬性——設計文件 7.3）。
3. Checkbox 可點擊區 ≥ 44×44px；label 點擊亦可切換。
4. 若使用者未勾選而嘗試點擊，顯示文字提示（不得只靠紅框）：
   `請先閱讀並勾選同意，才能查看結果`。

- [ ] 告知文案一字不差；checkbox 預設未勾
- [ ] 未勾選無法送出（含鍵盤操作路徑）；勾選後流程與現況完全一致
- [ ] 評估計分與寫入行為零變更（比對送出 payload 一致）
- [ ] 三端 smoke；`npm run build` 通過
- [ ] 回報時附截圖供 Specialist 過目文案
